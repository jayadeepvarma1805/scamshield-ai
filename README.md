# ScamShield AI

An AI-powered web application designed to detect and analyze potential online scams, phishing attempts, and suspicious URLs. ScamShield helps users identify fraudulent content using machine learning techniques and an intuitive user interface.

## Features

- AI-powered scam detection
- Suspicious URL analysis
- Phishing detection
- Threat assessment dashboard
- Responsive web interface
- Secure backend API
- Fast threat analysis

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

### Backend
- Node.js
- Express.js
- JavaScript
